Autonomous AI Agent System with Root Access

🤖

Agent Loop Architecture

Proper planning-acting-observing loop with structured JSON outputs. The AI doesn't just respond once - it plans multi-step tasks and executes them autonomously.

⚡

Lightning Fast

Powered by Groq LLaMA 3.3 70B with 15-second timeout and automatic failover to OpenRouter. No more waiting forever for responses.

🧠

Memory System

Persistent short-term and long-term memory stored in JSON. The AI remembers your conversations and learns from previous interactions.

🛠️

Tool System

Safe tool-based execution - run commands, read/write files, list directories, open browsers. No blind root access - everything goes through defined tools.

🌐

Beautiful Web UI

FastAPI-powered web interface with real-time WebSocket updates. Dark theme, responsive design, execution logs, and tool quick-access.

⌨️

Hotkey Summon

Press Ctrl+Shift+A anywhere to summon the overlay. Desktop integration with system tray icon and global hotkeys.

Quick Start

# Clone the repository
git clone https://github.com/ANIBOY001/AryanAssists.git
cd AryanAssists

# Install dependencies
pip install -r requirements.txt

# Run Web UI
python main.py --mode web

# Or CLI mode
python main.py --mode cli

# Or Desktop Overlay (Ctrl+Shift+A)
python main.py --mode overlay
